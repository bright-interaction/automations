## Import Instructions

1. Open Make dashboard
2. Click **"Create a new scenario"**
3. Click **"..."** menu → **"Import Blueprint"**
4. Upload the `.json` file
5. Reconnect all modules to your accounts
6. Set scheduling and activate

## File Format

- Export via **"..."** → **"Export Blueprint"**
- Blueprints exclude credentials and data store contents

## Operations & Limits
